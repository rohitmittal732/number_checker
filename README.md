# Positive-Negative Checker (Python)

This is a simple Python program that checks whether a given number is positive, negative, or zero.  
The user enters any number, and the program displays the result using basic if-else conditions.

## Features
- Checks if the number is positive
- Checks if the number is negative
- Checks if the number is zero
- Handles both integers and decimal numbers
